# A2UI Financial Assistant

An AI-powered financial advisory assistant that dynamically generates user interfaces using the A2UI specification.
## 🚀 Tech Stack

### Backend
- FastAPI
- Python
- Google Gemini
- Redis

### Frontend
- React
- TypeScript
- Tailwind CSS

## 📌 Features

- AI-powered stock comparison
- Dynamic A2UI rendering
- Conversation memory
- Streaming responses
- Personalized investment recommendations
